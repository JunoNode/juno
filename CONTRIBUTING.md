# Contributing to Juno

Thank you for considering contributing!

## Setup

1. Clone the repo
2. Run `npm install`
3. Start dev server: `npm run dev`

## Folder Structure
- `src/`: App logic
- `lib/`: API and utility logic
- `features/`: Scoped features (chat, settings, etc.)

## Contribution Flow
- Branch from `dev`
- PR into `main` only after review
